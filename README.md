# EnglishBoost - Landing Page Website

## 📌 Deskripsi Project
EnglishBoost adalah website landing page bertema pembelajaran Bahasa Inggris yang dirancang dengan desain modern dan responsif. Website ini dibuat menggunakan HTML, CSS, dan sedikit JavaScript untuk fitur interaktif.

## 🎯 Tujuan Pembuatan
Project ini dibuat sebagai tugas untuk menampilkan kemampuan dalam:
- Membuat struktur HTML
- Mendesain tampilan menggunakan CSS
- Menambahkan interaktivitas menggunakan JavaScript
- Mengupload dan deploy project menggunakan GitHub Pages

## 🛠️ Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript
- GitHub Pages (Deployment)

## ✨ Fitur Website
- Tampilan modern dengan gradient background
- Responsive design (mobile friendly)
- Hover effect pada fitur
- Form registrasi dengan notifikasi sukses
- Deploy online menggunakan GitHub Pages
